# НЕ ТРОГАТЬ ЭТУ ВЕТКУ!!!

В `infrastructure/argocd/infra-apps.yaml` лежит манифест App-of-Apps, он
как раз отслеживает изменения в этой ветке

Так как деплоить можно из любой ветки, а ArgoCD может отслеживать
изменения только в одной ветке, то для удобной, корректной работы
нужна ветка, которая будет жить на протяжении всего проекта и в нее
можно безпрепятственно пушить из CI
