# Fix 400 "invalid schema" на UpdateTimeLine

Логирование + валидация. Только 1 файл.

Файл: `facade/internal/delivery/http/handlers/card.go` (функция `UpdateTimeLine`, строки 1209-1217)

**Было:**
```go
var req dto.NewTimeLine
if err := easyjson.UnmarshalFromReader(r.Body, &req); err != nil {
    api.RespondError(w, http.StatusBadRequest, handlerCommon.ErrInvalidRequestSchema.Error())
    return
}

if req.Start.After(req.DeadLine) {
    api.RespondError(w, http.StatusBadRequest, handlerCommon.ErrSetTimeLine.Error())
    return
}
```

**Стало:**
```go
var req dto.NewTimeLine
if err := easyjson.UnmarshalFromReader(r.Body, &req); err != nil {
    logger.Warn().Err(err).Msg("UpdateTimeLine invalid JSON body")
    api.RespondError(w, http.StatusBadRequest, handlerCommon.ErrInvalidRequestSchema.Error())
    return
}

if req.Start.IsZero() || req.DeadLine.IsZero() {
    api.RespondError(w, http.StatusBadRequest, "start and deadline are required")
    return
}
if req.Start.After(req.DeadLine) {
    logger.Warn().Time("start", req.Start).Time("deadline", req.DeadLine).Msg("start after deadline")
    api.RespondError(w, http.StatusBadRequest, "start must be before or equal to deadline")
    return
}
```
